# Admin Specification: Exercises

## Overview
This document outlines the specifications for the Exercises management screen in the Admin portal.

## Key Features
- Manage the global library of exercises
- Add, edit, or remove exercises
- Categorize exercises by muscle group or type

## UI Components
- Exercise Library Grid/Table
- "Add New Exercise" Modal
- Category Filters

## Data Requirements (API)
- `GET /api/admin/exercises` - Fetch global exercise library
- `POST /api/admin/exercises` - Add new exercise

## Future Enhancements
- Video upload for exercise demonstrations
